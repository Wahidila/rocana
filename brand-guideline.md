## 2. Logo System — Four Variants

The identity uses 4 logo variants. Always use the correct one for each context.

### 2.1 Logo Variants

| Variant | File | Description | Color |
|---------|------|-------------|-------|
| **Logo Main** | `rocana studio logo main.png` | Full logo: spiral symbol + "ROCANA.STUDIO" text stacked below | Sage Olive `#8B8660` |
| **Logo Black** | `rocana studio logo black.png` | Full logo: same layout, solid black | Obsidian Black `#000000` |
| **Logo Mark** | `rocana studio logo mark.png` | Symbol only: spiral "R" without text | Sage Olive `#8B8660` |
| **Logotype** | `rocana studio logotype.png` | Text only: "ROCANA.STUDIO" without symbol | Sage Olive `#8B8660` |

### 2.2 Logomark Anatomy

- Organic spiral circle resembling water vortex / fingerprint / natural element
- Letter "R" seamlessly integrated into center of spiral
- Deliberately rough, imperfect outer edges — handmade, earthy, Wabi-sabi aesthetic
- Spiral flows inward — symbolizes capturing the soul within

### 2.3 Logotype Anatomy

- Font: bold, rounded sans-serif (Nunito Black style)
- "ROCANA.STUDIO" — period acts as deliberate pause between brand name and craft
- Rounded letterforms echo organic curves of the spiral symbol
- Always uppercase

### 2.4 Usage Matrix

| Context | Recommended Variant | Notes |
|---------|-------------------|-------|
| Website hero / header | Logo Main | Sage Olive on light; invert for dark |
| Navigation bar | Logo Mark + Logotype | Symbol left, text right — horizontal |
| Business cards / Print | Logo Black | Maximum contrast for print |
| Social media avatar | Logo Mark | Symbol only at small sizes |
| Photo watermark | Logotype | Text-only at 15–25% opacity |
| Email signature | Logotype | Compact horizontal, Sage Olive |
| Favicon / App icon | Logo Mark | Works at 16px–512px |
| Proposals / Invoices | Logo Main | Full logo top of document |

### 2.5 Clear Space

Minimum clear space around any logo variant = width of the "R" in the logomark. No other elements may intrude into this zone.

### 2.6 Minimum Size

| Variant | Print | Digital |
|---------|-------|---------|
| Logo Main | 30mm width | 120px width |
| Logo Black | 30mm width | 120px width |
| Logo Mark | 10mm width | 32px width |
| Logotype | 25mm width | 100px width |

### 2.7 Logo Rules

**Do:**
- Use correct variant for each context (see usage table)
- Maintain required clear space
- Use inverted (white) version on dark/busy backgrounds
- Use Logo Black for monochrome/high-contrast needs
- Use Logo Mark alone for small sizes (avatars, favicons)
- Use Logotype alone for horizontal compact layouts

**Don't:**
- Stretch, skew, or distort any variant
- Change colors outside approved palette
- Add drop shadows, outlines, glows, or effects
- Place on busy backgrounds without sufficient contrast
- Rotate or rearrange symbol and text
- Recreate logotype using alternative fonts
- Combine Logo Mark with a different typeface
- Use full Logo Main below minimum size threshold

## 3. Color Palette — Earth & Soul

Inspired by Bali's volcanic earth, ancient temple stones, and sun-bleached coastal sands.

### 3.1 Brand Colors

| Name | Hex | Role |
|------|-----|------|
| **Sage Olive** | `#8B8660` | **Primary brand color** — logo color, organic, calm, premium |
| **Obsidian Black** | `#000000` | Logo alternative, text, professional, elegant |
| **Sand Cream** | `#E5E2D1` | Background — clean, warm, breathing space |
| **Deep Moss** | `#4A4D39` | Contrast — depth, grounding, emphasis |
| **Warm Stone** | `#A69F8A` | Secondary — subtle, supportive, neutral |
| **Ivory White** | `#F5F3EC` | Canvas — light background, whitespace |
| **Charcoal** | `#2C2C2C` | Text — softer alternative to pure black |
| **Muted Gold** | `#B8A88A` | Highlight — luxury accent, foil printing |

### 3.2 Color Usage Ratio

| Color | Ratio |
|-------|-------|
| Ivory White | 40% |
| Sand Cream | 25% |
| Sage Olive | 20% |
| Obsidian Black | 10% |
| Deep Moss | 5% |

### 3.3 Color Rules

**Do:**
- Sage Olive logo on Ivory White or Sand Cream backgrounds (primary brand look)
- Obsidian Black text on light backgrounds for readability
- Deep Moss for secondary text and supporting elements
- Muted Gold sparingly for luxury touches in print / foil stamping
- Inverted (white) logo on Obsidian Black for dark-mode / cinematic contexts

**Don't:**
- Sage Olive as background for body text (insufficient contrast)
- Warm Stone text on Sand Cream (too low contrast)
- Bright, saturated colors outside this palette
- Gradients mixing brand colors with non-palette colors
- Sage Olive logo on medium-toned backgrounds where it disappears

## 4. Typography

Three typefaces in harmony: elegant serif, bold rounded sans-serif (matching logo), clean body font.

### 4.1 Font Stack

| Role | Font | Style | Use For |
|------|------|-------|---------|
| **Display** | Cormorant | Serif, 400–700, italic | Hero headlines, section titles, artistic statements |
| **Heading** | Nunito | Rounded sans-serif, 800–900 | Navigation, labels, UI elements, logo companion |
| **Body** | Inter | Sans-serif, 300–600 | Paragraphs, proposals, captions, extended text |

**Why Nunito:** Rounded terminals echo the organic curves of the spiral logomark. Creates visual consistency between logo and supporting typography.

### 4.2 Type Scale

| Size | Use |
|------|-----|
| 48px / 3rem | Display |
| 32px / 2rem | Heading 1 |
| 24px / 1.5rem | Heading 2 |
| 18px / 1.125rem | Heading 3 |
| 16px / 1rem | Body text (minimum) |
| 14px / 0.875rem | Label / Nav |
| 11px / 0.688rem | Caption / Meta |

### 4.3 Typography Rules

- Nunito headings: always uppercase, letter-spacing 0.06em
- Body text: minimum 16px, line-height 1.6–1.8
- Cormorant display: use italic for artistic emphasis
- Max line length: 65–75 characters

### 4.4 Google Fonts Import

```css
@import url('https://fonts.googleapis.com/css2?family=Cormorant:ital,wght@0,400;0,500;0,600;0,700;1,400;1,500&family=Nunito:wght@400;500;600;700;800;900&family=Inter:wght@300;400;500;600&display=swap');

--font-display: 'Cormorant', Georgia, serif;
--font-heading: 'Nunito', 'Helvetica Neue', sans-serif;
--font-body: 'Inter', 'Helvetica Neue', sans-serif;
```

## 5. Visual Style & Tone of Voice

### 5.1 Visual Aesthetic

**Style:** Nature Distilled — muted earthy tones, organic textures, warm shadows, luxurious yet grounded.

**Design Principles:**

| Principle | Description |
|-----------|-------------|
| Whitespace | Let the work breathe. Generous margins, gallery-like experience. |
| Contrast | Bold black vs soft cream. Delicate serif vs rounded sans-serif. |
| Texture | Subtle grain, natural paper textures, organic shapes for depth. |
| Restraint | Less is more. Every element must earn its place. |

### 5.2 Tone of Voice

| Tone | Description | Example |
|------|-------------|---------|
| Sophisticated | Quiet confidence. No exclamation marks or hyperbole. | "A morning ceremony in Ubud, where incense met the first light and two souls became one." |
| Artistic | Evocative, sensory language. Paint pictures with words. | "We found them at golden hour, barefoot on volcanic sand, the ocean writing their story in foam." |
| Professional | Clear and direct for business. Warm but efficient. | "Your gallery of 300+ edited images will be delivered within 21 working days via a private online gallery." |
| Storytelling | Every caption tells a micro-story. Focus on "why" not "what." | "She didn't know he was watching. That's when we pressed the shutter — in the space between awareness and abandon." |

### 5.3 Language

- **Primary:** English (British spelling: colour, favourite, centre)
- **Use words:** capture, reveal, craft, soul, light, story, moment, authentic, intimate, timeless
- **Avoid words:** cheap, discount, snap, pic, selfie, package deal, basic, standard, budget

## 6. Photography Style

### 6.1 Visual Characteristics

| Aspect | Guideline |
|--------|-----------|
| **Warm Tones** | Rich golden warmth. Skin tones glow. Shadows lean amber, not blue. |
| **Deep Shadows** | Embrace darkness for mystery and depth. Not crushed — rich with detail. |
| **Natural Texture** | Focus on skin, fabric, stone, water, leaves. Macro details. |

### 6.2 Color Grading Reference

Warm Highlight → Mid Tone → Rich Shadow → Deep Black
`#D4A574` → `#8B7355` → `#5C4A3A` → `#3A2E24` → `#1A1410`

### 6.3 Composition Rules

- Prioritize negative space — let subjects breathe
- Use natural framing: doorways, foliage, architecture
- Shoot at eye level or slightly below for intimacy
- Include environmental context (Bali landscapes, temples, ocean)
- Candid over posed — capture the in-between moments

### 6.4 Post-Processing

- Warm white balance: 5800K–6500K
- Slightly desaturated — never oversaturated
- Lift blacks slightly for filmic quality
- Subtle grain: 15–25% in Lightroom
- Consistent color grading across entire galleries

## 7. Implementation

### 7.1 Watermark

- Use **Logotype** (text-only) or **Logo Mark** (symbol-only) as watermarks
- Never use full Logo Main as watermark (too complex at low opacity)
- Opacity: 15–25%
- Position: bottom-right or bottom-center
- White on dark images, Sage Olive on light images

### 7.2 Instagram Grid

- Minimalist, curated grid
- Pattern: Photo → Reel → Photo → Quote/BTS → Repeat
- Consistent warm toning across all posts
- Profile avatar: Logo Mark (symbol only)
- Cinematic color grading on Reels

### 7.3 Brand Applications

| Application | Logo Variant | Details |
|-------------|-------------|---------|
| **Website** | Nav: Mark + Logotype; Hero: Logo Main; Footer: Logo Main inverted | Dark hero sections, generous whitespace, lazy-loaded images |
| **Business Cards** | Front: Logo Mark (foil); Back: Logotype + contact | 400gsm uncoated, soft-touch lamination, optional deboss |
| **Proposals** | Page 1: Logo Main; Footer: Logotype | Ivory White bg, Nunito headers, Inter body 11pt, Sage Olive accents |
| **Email Signature** | Logotype (Sage Olive) | Nunito Bold name, Inter details, Cormorant italic tagline |
